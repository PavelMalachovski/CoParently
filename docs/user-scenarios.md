# Пользовательские сценарии (MVP)

## 1. Совместный календарь для родителей
**Описание:** Два родителя (или опекуна) могут видеть единый календарь с событиями, связанными с уходом за ребёнком.

**Сценарии использования:**
- Родитель A создаёт событие "Встреча с учителем"
- Родитель B видит это событие в своём календаре
- Оба родителя видят, у кого ребёнок в конкретный день

## 2. Распределение дней/недель опеки
**Описание:** Родители могут настроить регулярный график опеки (например, понедельник-среда у мамы, четверг-воскресенье у папы).

**Сценарии использования:**
- Родитель A настраивает график опеки (например, "Каждую неделю: понедельник-среда у мамы")
- Приложение автоматически создаёт события для каждого периода
- Оба родителя видят график опеки в календаре

## 3. Добавление событий
**Описание:** Родители могут создавать события, связанные с уходом за ребёнком.

**Типы событий:**
- "Сегодня у мамы" / "Сегодня у папы"
- "Врач" (дата и время приёма)
- "Тренировка" (регулярное или разовое мероприятие)
- "Школа" (расписание занятий)
- "День рождения" / "Праздник"
- Произвольное событие с описанием

**Сценарии использования:**
- Родитель A создаёт событие "Тренировка футбола" на каждый вторник
- Родитель B редактирует событие, добавляя комментарий
- Родитель A удаляет устаревшее событие

## 4. Уведомления и синхронизация с Google Calendar
**Описание:** События из CoParently синхронизируются с Google Calendar пользователя.

**Сценарии использования:**
- После создания события в CoParently оно автоматически появляется в Google Calendar
- Изменение события в Google Calendar синхронизируется обратно в CoParently
- Пользователь получает push-уведомления о новых событиях, созданных другим родителем
- Пользователь может отключить синхронизацию в настройках

## Приоритеты для MVP
1. **Высокий приоритет:**
   - Просмотр календаря с событиями
   - Создание и редактирование событий
   - Цветовая маркировка событий по родителю

2. **Средний приоритет:**
   - Настройка графика опеки
   - Синхронизация с Google Calendar

3. **Низкий приоритет (для следующих этапов):**
   - Push-уведомления
   - Совместное редактирование в реальном времени
   - Чат между родителями
