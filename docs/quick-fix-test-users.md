# Быстрое решение: Добавление тестовых пользователей

## Проблема

Вы видите сообщение о том, что приложение в режиме тестирования и доступно только для тестеров.

## Быстрое решение (5 минут)

### Шаг 1: Откройте OAuth Consent Screen

1. Перейдите: https://console.cloud.google.com/apis/credentials/consent
2. Выберите проект: **coparently-a39c9**

### Шаг 2: Добавьте тестовых пользователей

1. Прокрутите вниз до раздела **Test users**
2. Нажмите **+ ADD USERS**
3. Введите email адреса (через запятую или по одному):
   ```
   ваш-email@gmail.com
   тестовый-email@gmail.com
   ```
4. Нажмите **ADD**

### Шаг 3: Проверьте приложение

1. Используйте Google аккаунт, который добавлен в список
2. Запустите приложение
3. Попробуйте войти через Google Sign-In
