# Прототипы экранов

## 1. Главный календарь
**Описание:** Основной экран приложения, отображающий календарь с событиями.

**Компоненты:**
- Календарь (месячный вид с возможностью переключения на недельный)
- Список событий на выбранный день
- FAB (Floating Action Button) для добавления нового события
- Индикатор текущего дня
- Цветовая маркировка: розовый для "Мама", синий для "Папа"

**Навигация:**
- Тап на событие → открывает экран редактирования события
- Тап на день → показывает список событий на этот день
- Swipe для переключения месяца/недели

## 2. Список событий
**Описание:** Детальный список всех событий с фильтрацией.

**Компоненты:**
- Список событий (сгруппированные по датам)
- Фильтр по родителю (все / мама / папа)
- Поиск по названию события
- Сортировка (по дате, по типу события)

**Навигация:**
- Тап на событие → открывает экран редактирования события
- Long press на событие → показывает контекстное меню (удалить, дублировать)

## 3. Экран добавления/редактирования события
**Описание:** Форма для создания или редактирования события.

**Поля формы:**
- Название события (обязательное)
- Дата и время начала
- Дата и время окончания (опционально)
- Тип события (выбор из списка)
- Родитель (мама / папа / оба)
- Описание/комментарий (опционально)
- Повторяющееся событие (ежедневно, еженедельно, ежемесячно)

**Компоненты:**
- Date/Time Picker для выбора даты и времени
- Dropdown для выбора типа события
- Radio buttons для выбора родителя
- Checkbox для настройки повторения
- Кнопки "Сохранить" и "Отмена"

## 4. Настройки пользователя
**Описание:** Экран с настройками приложения и профилем пользователя.

**Разделы:**
- **Профиль:**
  - Имя родителя
  - Email
  - Фото профиля (опционально)
  - Цвет для отображения в календаре (розовый/синий)

- **График опеки:**
  - Настройка регулярного графика (например, понедельник-среда у мамы)
  - Сохранение и редактирование шаблонов

- **Google Calendar:**
  - Переключение синхронизации
  - Выбор календаря для синхронизации
  - Статус подключения

- **Приглашение второго родителя:**
  - Отправка приглашения по email
  - Поделиться ссылкой-приглашением

- **Общие настройки:**
  - Язык приложения
  - Тема (светлая/тёмная)
  - Уведомления

**Навигация:**
- Возврат на главный экран
- Сохранение изменений

## Цветовая схема (Material 3)
- **Мама:** Pink (Material 3 Pink palette)
- **Папа:** Blue (Material 3 Blue palette)
- **Общие события:** Gray или комбинация цветов
- **Фон:** Системная тема (светлая/тёмная)
- **Акценты:** Primary цвет из Material 3
